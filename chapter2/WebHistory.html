웹의 역사

1. 월드 와이드 웹의 탄생

1989년, 팀 버너스리(Tim Berners-Lee)는 유럽 입자 물리 연구소(CERN)에서 정보를 공유하기 위한 시스템으로 월드 와이드 웹(World Wide Web, WWW)을 제안하였습니다. 1990년에는 최초의 웹 브라우저와 웹 서버가 개발되었습니다.

2. 웹의 발전

1993년, 모자이크(Mosaic)라는 그래픽 기반 웹 브라우저가 등장하면서 웹은 대중적으로 확산되기 시작했습니다. 이후 넷스케이프, 인터넷 익스플로러 등 다양한 브라우저가 출시되었습니다.

3. 웹 2.0의 등장

2000년대 중반, 사용자 참여와 소통이 강조된 웹 2.0 시대가 열렸습니다. 블로그, 위키, 소셜 네트워크 서비스(SNS) 등이 대표적입니다.

4. 모바일 웹과 현대 웹

스마트폰의 보급과 함께 모바일 웹이 발전하였고, 반응형 웹 디자인, HTML5 등 새로운 기술이 도입되었습니다. 현재 웹은 다양한 기기와 플랫폼에서 사용되고 있습니다.
